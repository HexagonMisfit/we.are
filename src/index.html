VJ Tool - minim

Real-time 2D mograph system for doing live visuals for music. The concept is to trigger black-and-white animated 2D shapes, pass them through a gradient color mapper, then through some custom post processing effects, put them on a background, and then map a midi controller to trigger the different clips and alter the parameters.

Who Are We?

Based in rainy Seattle, Washington, we are an interaction design studio that uses technology to build experiences that astonish and delight. Our specialty is that we all have different specialties! Hit us up if you wanna collaborate on something dope.
